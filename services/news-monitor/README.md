# News Monitor — Total.kz

Автоматический мониторинг новостных источников для Total.kz.

## Что делает

1. **Мониторит RSS-ленты** казахстанских и мировых источников каждые 3-15 минут
2. **Переписывает статьи** через GPT-4o-mini в редакционном стиле Total.kz (на русском)
3. **Публикует в CMS** со статусом `review` (не автоматическая публикация)
4. **Уведомляет редакцию** в Telegram с кнопками управления

## Запуск

### В Docker (рекомендуется)

```bash
docker compose up -d news-monitor
```

### Dry-run (тестовый режим)

Проверяет источники, показывает что будет обработано, без вызова GPT и записи в БД:

```bash
docker exec total_kz_monitor python monitor.py --dry-run
```

### Переменные окружения

| Переменная | Обязательна | Описание |
|-----------|-------------|----------|
| `PG_DATABASE_URL` | Да | PostgreSQL URL |
| `OPENAI_API_KEY` | Да | Ключ OpenAI API |
| `TELEGRAM_BOT_TOKEN` | Нет | Токен Telegram бота |
| `TELEGRAM_CHAT_ID` | Нет | ID чата для уведомлений |

## Архитектура

```
monitor.py  → Основной цикл, проверка источников
rewriter.py → Переписывание через GPT-4o-mini
publisher.py → Запись в PostgreSQL
notifier.py → Уведомления в Telegram
sources.yaml → Конфигурация источников
```

Дедупликация URL через SQLite (`/app/data/seen_urls.db`).

## Telegram-кнопки

- **✅ Опубликовать** — меняет статус на `published`
- **📝 В редактор** — открывает в CMS-редакторе
- **❌ Пропустить** — помечает как `rejected`
